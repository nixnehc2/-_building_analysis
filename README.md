# 建筑轮廓形态分析工具集

本项目包含一系列用于建筑轮廓形态分析的 Python 脚本，主要用于从 JSON 数据中提取建筑边界信息，计算各种形态指标，并进行数据处理和可视化。

## 环境依赖

```bash
pip install pandas numpy shapely openpyxl pyproj matplotlib
```

## 程序说明

### 1. caculate.py - 基础边界计算工具

**功能**：批量处理 JSON 文件，计算建筑边界的基本几何属性。

**主要功能**：
- 使用鞋带公式计算多边形面积
- 计算多边形周长
- 支持像素、毫米、米等多种单位转换
- 批量处理目录下所有 JSON 文件
- 输出 Excel 和 CSV 格式的汇总表格

**输出指标**：周长、面积（像素/毫米/米单位）

---

### 2. caculate1.py - 完整形态分析工具

**功能**：全面的建筑轮廓形态分析，计算多种形态指标。

**主要功能**：
- 计算长宽比和主方向角（基于最小外接矩形）
- 计算转角密度（有效转角数/周长）
- 计算紧凑度（圆形度）：`(4πA) / P²`
- 计算凹凸度：轮廓面积/凸包面积
- 自动生成指标说明文档

**输出指标**：
- 基本几何：周长、面积、凸包面积、最小矩形面积
- 形态指标：长宽比、主方向角、凹凸度、紧凑度、转角密度
- 衍生指标：面积利用率、凸性指标

---

### 3. caculate2.py - GIS 数据处理工具（简化版）

**功能**：处理 GIS 格式的 JSON 数据（包含 features 数组）。

**主要功能**：
- 解析 GIS 标准格式的 JSON 数据
- 提取建筑属性（FID、高度、年份等）
- 计算形态指标
- 支持南京地区纬度投影修正（可选）

**输出字段**：FID、建筑高度、建筑年份、周长、面积、长宽比、主方向角、凹凸度、紧凑度、转角密度

---

### 4. caculate3.py - GIS 数据处理工具（坐标转换版）

**功能**：带坐标系转换的 GIS 数据处理。

**主要功能**：
- 使用 pyproj 进行坐标系转换（EPSG:3857 → EPSG:4507）
- 支持 CGCS2000 投影坐标系
- 输出高精度米制单位数据

**适用场景**：需要精确地理坐标的建筑分析

---

### 5. caculate4.py - 带图结构分析的形态工具

**功能**：除形态分析外，还支持建筑空间关系图的生成。

**主要功能**：
- 支持自定义转角参数（角度阈值、边长比阈值）
- 分析房间和门的空间关系
- 生成空间邻接矩阵
- 检测多边形交集关系

**特色**：可用于建筑内部空间拓扑分析

---

### 6. caculate5.py - Residence 建筑专用分析工具

**功能**：专门处理住宅类型（Residence）建筑的形态分析。

**主要功能**：
- 筛选 Function 为 "Residence" 的建筑
- 坐标点简化（合并过近的点）
- 多边形可视化并保存为 PNG 图片
- 输出中间计算结果

**输出文件**：
- `建筑形态分析_Residence.xlsx` - 最终结果
- `建筑形态中间信息_Residence.xlsx` - 中间数据
- `polygon_visualizations/` - 可视化图片目录

---

### 7. caculate6.py - 多边形修复工具

**功能**：对建筑轮廓多边形进行修复和规整化处理。

**主要功能**：
- 对称性检测与修复（垂直/水平对称）
- 直角修正（将接近90°的角度调整为精确直角）
- 坐标点简化（删除过近的点）
- 角度简化（删除角度过小的顶点）
- 可视化修复前后的对比

**输出**：修复后的 JSON 文件保存在 `repair/` 目录

---

### 8. draw1.py - 多边形简化与可视化工具

**功能**：使用 Douglas-Peucker 算法简化多边形并进行规整化。

**主要功能**：
- 多边形简化（可调节容差参数）
- 将斜边转换为水平/垂直边
- 删除角度过小的顶点
- 可视化处理结果

**使用方法**：将 JSON 文件放入 `repair/` 目录，运行脚本进行处理

---

## 数据格式说明

### 输入 JSON 格式（户型图数据）

```json
{
  "floor1": {
    "meta": {
      "name": "楼层名称",
      "lengthRate": "0.100 mm/pixel"
    },
    "boundary": {
      "边界": [["(x1,y1)", "(x2,y2)", ...]]
    },
    "rooms": { ... },
    "doors": { ... }
  }
}
```

### 输入 JSON 格式（GIS 数据）

```json
{
  "features": [
    {
      "attributes": {
        "FID": 1,
        "Height": 10.5,
        "Age": 2000,
        "Function": "Residence"
      },
      "geometry": {
        "rings": [[[x1, y1], [x2, y2], ...]]
      }
    }
  ]
}
```

## 形态指标说明

| 指标 | 说明 | 取值范围 |
|------|------|----------|
| 长宽比 | 最小外接矩形的长/宽 | ≥1 |
| 主方向角 | 长边与正东方向的夹角 | 0-180° |
| 紧凑度 | 形状接近圆形的程度 | 0-1 |
| 凹凸度 | 轮廓面积/凸包面积 | 0-1 |
| 转角密度 | 有效转角数/周长 | ≥0 |

## 使用示例

```python
# 运行基础计算
python caculate.py

# 运行完整形态分析
python caculate1.py

# 处理 GIS 数据
python caculate2.py

# 修复多边形
python caculate6.py
```

## 注意事项

1. 运行前请确保 JSON 文件与脚本在同一目录下
2. 部分脚本需要用户输入参数（如转角阈值）
3. 输出文件会覆盖同名文件，请注意备份
4. 可视化功能需要图形界面支持
